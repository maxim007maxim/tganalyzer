# TG Analyzer Bot

Telegram-бот для анализа каналов и расчёта справедливой цены рекламы.

## Запуск
```bash
pip install -r requirements.txt
BOT_TOKEN=ваш_токен python bot.py
```

## Что умеет
- Анализ подписчиков, охвата, ER
- Расчёт справедливой цены рекламы по CPM
- Определение накрутки по низкому ER
- Лимит 3 проверки/день для бесплатных пользователей
